

The code for this project was written in Python 3.6 and was done on jupyter notebook. The files are all in the folder "jpatel345" 

To run this, while being in the correct directory, run
'jupyter notebook' in terminal to launch the notebook.

Once the notebook is open you can click on the 'Kernel' tab and click 'Restart and Run All"

Make sure you have all the dependencies if you do not already have them.
